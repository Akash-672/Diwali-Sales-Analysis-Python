# Diwali Sales Analysis | Python EDA Project

## Project Overview

This project performs Exploratory Data Analysis (EDA) on Diwali sales data to identify customer purchasing behavior and sales trends.

The objective is to understand which customer segments contribute the most to sales and help businesses improve marketing and sales strategies.

---

## Dataset

The dataset contains customer transaction data, including:

* Gender
* Age Group
* State
* Marital Status
* Occupation
* Product Category
* Orders
* Sales Amount

---

## Project Tasks

* Data Cleaning
* Handling Missing Values
* Data Type Conversion
* Data Exploration
* Customer Segmentation
* Sales Analysis
* Data Visualization

---

## Analysis Performed

### Gender Analysis

* Compared purchase behavior of male and female customers.
* Analyzed sales contribution by gender.

### Age Group Analysis

* Identified the age groups contributing the highest sales.

### State-wise Analysis

* Analyzed top states based on orders and sales amount.

### Marital Status Analysis

* Compared purchasing patterns based on marital status.

### Occupation Analysis

* Identified occupations with the highest purchasing activity.

### Product Category Analysis

* Analyzed top-performing product categories based on sales.

---

## Key Insights

* Female customers contributed more sales than male customers.
* Customers aged **26–35 years** generated the highest sales.
* Uttar Pradesh, Maharashtra, and Karnataka were the top-performing states.
* Married women contributed a significant share of total purchases.
* Customers working in IT, Healthcare, and Aviation sectors spent the most.
* Food, Clothing, and Electronics were the most purchased product categories.

## Conclusion

The analysis shows that married women aged 26–35 years from Uttar Pradesh, Maharashtra, and Karnataka working in IT, Healthcare, and Aviation sectors are the most likely customers to purchase products, particularly in Food, Clothing, and Electronics categories.
